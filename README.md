## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ruby: 2.7.1
Rails: 6.0.3.2
Postgres: >=9.5

### Setup

Install gems with:

```
    bundle install
```

install dependencies
```
    yarn
```

Setup database with:

```
    rails db:create
    rails db:migrate
```



### Usage

Start server with:

```
    rails s
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    bundle exec rspec
```
